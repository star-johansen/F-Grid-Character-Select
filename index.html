Manage Profiles

Hotkeys:

• Ctrl/Cmd + 1-9: Quick switch to profile

• Ctrl/Cmd + Left/Right: Navigate between profiles

• Ctrl/Cmd + P: Open profile manager

• Each profile maintains its own characters, favorites, and settings

Help & Keyboard Shortcuts

Keyboard Shortcuts

General

Ctrl/Cmd + H or Ctrl/Cmd + / - Show this help window
Esc - Close any open popup
Enter - Add character (when in input field)

Profile Management

Ctrl/Cmd + P - Open profile manager
Ctrl/Cmd + 1-9 - Quick switch to profile
Ctrl/Cmd + ←/→ - Navigate between profiles

Selection Mode

Ctrl/Cmd + A - Select all visible characters
Right Click - Toggle selection on character

Batch Operations

Ctrl/Cmd + Enter - Submit batch add (in batch dialog)

Mouse Actions

Left Click - Open character page / Select in selection mode
Middle Click - Open character in background tab
Right Click - Open character details / Toggle selection
Drag & Drop - Add character to favorites
Click Checkbox - Enable selection mode and select character

Features

• Multiple Profiles - Manage different character sets (e.g., your characters vs friends')
• Search - Normal or regex search with history
• Tags - Organize characters with custom tags
• Batch Import - Add multiple characters at once (comma/newline/tab separated)
• Auto-backup - Data is backed up every 5 minutes
• Export/Import - Save and restore all your data as JSON
• Themes - Choose between Default (blue), Dark (purple), or Light (orange)

Tips

• Character avatars are automatically fetched from F-List
• You can upload custom avatars by right-clicking a character
• Use regex search for advanced pattern matching
• The app works offline - all data is stored locally
• Search history shows your last 10 searches
